# Requerimientos

CLI que permita:

- todo add <texto>
- todo list
- todo done <id>
- todo delete <id>

# No funcionales

1. Sin almacenamiento
2. Output simple
3. Errores explícitos
4. Separación de responsabilidades 
    - Entrada/Parsing
    - Lógica
    - Output
